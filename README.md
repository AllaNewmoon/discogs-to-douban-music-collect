# Discogs → Douban Music Collect

一个 Chrome 插件，用于从 Discogs 自动采集唱片信息，
并自动填写豆瓣音乐「新建条目」

## ✨ 功能
- 一键从 Discogs Release页面 Collect
- 自动填写豆瓣唱片名 → 自动跳转 → 自动填完整信息
- 支持：
  - 曲目表（自动格式化）
  - Credits自动填写（Producer / Written-By / Vocals 等）
  - 发行日期（自动规范为 YYYY-MM-DD）
  - 封面自动下载
- 特殊优化：
  - 不误拆 & / And 艺人名
  - 去掉艺人编号 / 星号

## 🧩 安装方式（Chrome）
1. 下载本仓库（Code → Download ZIP）
2. 解压
3. 打开 Chrome，访问：
   chrome://extensions
4. 打开右上角「开发者模式」
5. 点击「加载已解压的扩展程序」
6. 选择解压后的文件夹

## 🚀 使用方法
1. 打开 Discogs 的 Release页面
2. 左上角点击 **Collect**
3. 自动打开豆瓣音乐新建条目
4. 所有信息（除流派，专辑类型，介质）已自动填写完成

## ⚠️ 注意
- 本插件不修改任何豆瓣数据，仅模拟人工填写
- 仅供学习与个人使用
